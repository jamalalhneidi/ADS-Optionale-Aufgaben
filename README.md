## ADS Optionale Aufgeben 

Solutions for the optional tasks of ADS module 

### Folder Structure

- src/
- ├── zettel_x/
- ├──├── Main.java  - Solution
- ├──├── README.md        - Task-related info
    
### Running Solutions:

Java Version: **21.0.9**

```
git clone https://github.com/jamalalhneidi/ADS-Optionale-Aufgaben.git
```

```
cd ADS-Optionale-Aufgaben #Make sure you're in this directory
````

```
javac -d bin -sourcepath src src/zettel_{x}/Main.java && java -cp bin zettel_{x}.Main #Change {x} to the number of Zettel
```
